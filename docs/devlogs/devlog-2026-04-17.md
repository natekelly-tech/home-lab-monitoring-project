## Key Concepts Learned
- **ISP port blocking** — residential providers commonly block incoming connections; tunneling bypasses this
- **Cloudflare Tunnel** — outbound tunnel from server to Cloudflare edge; real enterprise pattern
- **Dynamic DNS** — keeps a domain pointed at a home IP even when ISP changes it
- **Domain registrar vs DNS provider** — Porkbun registers the domain, Cloudflare manages DNS; separate concerns
- **Nameserver delegation** — pointing Porkbun nameservers to Cloudflare hands DNS control to Cloudflare
